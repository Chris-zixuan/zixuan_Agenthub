## 📖 使用指南

### 🎮 三种输入模式

**模式一：直接输入** 📝  
把简历内容粘贴进来，一键优化！适合快速修改。

**模式二：上传 Word 文档** 📤  
把 `.docx` 文件放进 `input/` 文件夹，轻松搞定！

**模式三：上传 PDF 文档** 📄  
把文本型 PDF 文件放进 `input/` 文件夹，支持解析提取。

### 🎯 进阶玩法：JD 匹配优化

想让简历更精准？系统会问你：  
**"需要补充岗位 JD 信息吗？"**  
选「是」→ 粘贴 JD 文本 → 系统自动对标优化！🎯

### 📦 输出成果

优化完成后，你会得到：
- `姓名_简历.docx` - 专业 Word 格式，A4 排版完美
- `姓名_简历.pdf` - 打印级 PDF，直接投递
- `姓名_简历质量检测报告.md` - 竞争力评分 + 改进建议

## 📊 职场竞争力评分体系

### 🏆 评级标准

| 得分 | 评级 | 状态 | 建议 |
|:---:|:---:|:---:|:---|
| 90-100 | 🌟 S | 超神！ | 冲顶级企业！ |
| 80-89 | 🚀 A | 很强！ | 一线岗位随便投 |
| 70-79 | 💪 B | 不错！ | 主流企业稳了 |
| 60-69 | ⚠️ C | 一般 | 需要针对性优化 |
| 50-59 | 🚨 D | 较弱 | 建议深度优化 |
| 0-49 | 🔴 E | 不足 | 重新梳理重构 |

### 📈 评分维度

- **项目业绩** (25%)：项目数量、角色、成果量化
- **工作成果** (25%)：量化成果、业务价值
- **个人技能** (15%)：技能匹配度、深度广度
- **资质证书** (10%)：证书含金量、语言能力
- **简历逻辑性** (10%)：职业发展、能力递进
- **简历合理性** (10%)：经验匹配、职责合理
- **简历衔接性** (5%)：时间衔接、领域过渡

## 🛠️ 技术亮点

- **多格式支持** 📄：支持 `.docx`、PDF（文本型）和纯文本三种输入模式
- **跨平台兼容** 🌍：Windows 使用 Word COM 对象生成 PDF，非Windows系统自动降级为 HTML 输出
- **智能降级** 🧠：PDF 生成失败时自动输出 HTML 文件，用户可手动转换
- **中文友好** 🀄：完美支持中文字体渲染（Microsoft YaHei、PingFang SC）
- **规则本地化** 📚：优化规则全部本地存储，不联网搜索，响应更快更安全
- **规则可定制** ✏️：可根据需求修改 `references/` 目录下的规则文档
- **专业排版** 📐：标准 A4 纸张尺寸，符合打印规范
