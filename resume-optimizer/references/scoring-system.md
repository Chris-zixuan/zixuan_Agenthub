# 职场竞争力评分体系

## 概述

本评分体系从7个维度对简历进行加权评分，总分100分，最终得出职场竞争力综合评级。

## 评分维度与权重

| 评估维度 | 权重 | 评估重点 |
|:---:|:---:|:---|
| 项目业绩 | 25% | 项目数量、规模、角色、成果量化、STAR完整性 |
| 工作成果 | 25% | 成果量化、显著性、行动描述、专业表达 |
| 个人技能 | 15% | 技能相关度、深度、广度、佐证 |
| 资质证书 | 10% | 证书含金量、相关性、语言能力 |
| 简历逻辑性 | 10% | 职业发展逻辑、能力递进性、内容一致性 |
| 简历合理性 | 10% | 经验年限匹配、岗位职责匹配、成果匹配 |
| 简历衔接性 | 5% | 时间衔接、领域衔接、技能衔接、角色衔接 |

## 综合评级标准

| 得分区间 | 评级 | 含义 | 行动建议 |
|:---:|:---:|:---|:---|
| 90-100 | S | 极具竞争力，简历质量优秀 | 可投递顶级企业和核心岗位 |
| 80-89 | A | 竞争力强，简历质量良好 | 可投递一线企业核心岗位 |
| 70-79 | B | 具备竞争力，简历质量中等偏上 | 可投递主流企业目标岗位 |
| 60-69 | C | 竞争力一般，简历质量中等 | 需要针对性优化提升 |
| 50-59 | D | 竞争力较弱，简历需要较大改进 | 建议深度优化后再投递 |
| 0-49 | E | 竞争力不足，简历需要全面优化 | 建议重新梳理和重构 |

## 各维度评分标准

### 项目业绩（权重25%）

| 得分区间 | 评级 | 判断标准 |
|:---:|:---:|:---|
| 85-100 | 优秀 | 3个以上典型项目，均担任核心角色，成果全部量化，STAR结构完整 |
| 70-84 | 良好 | 2-3个典型项目，多数担任核心角色，多数成果有量化 |
| 55-69 | 中等 | 1-2个项目，角色描述模糊，部分成果有量化 |
| 40-54 | 较弱 | 无项目或项目描述简略，无量化成果 |
| 0-39 | 不足 | 无项目经历 |

### 工作成果（权重25%）

| 得分区间 | 评级 | 判断标准 |
|:---:|:---:|:---|
| 85-100 | 优秀 | 所有工作经历均有量化成果，成果显著且有业务价值 |
| 70-84 | 良好 | 多数工作经历有量化成果，成果较显著 |
| 55-69 | 中等 | 部分工作经历有成果描述，成果一般 |
| 40-54 | 较弱 | 工作经历描述模糊，缺少具体成果 |
| 0-39 | 不足 | 工作经历无法判断实际贡献和能力 |

### 个人技能（权重15%）

| 得分区间 | 评级 | 判断标准 |
|:---:|:---:|:---|
| 85-100 | 优秀 | 技能与目标岗位高度匹配，有深度有广度，均有项目经验佐证 |
| 70-84 | 良好 | 技能与岗位较匹配，有一定深度，多数有佐证 |
| 55-69 | 中等 | 技能与岗位部分匹配，深度一般，部分有佐证 |
| 40-54 | 较弱 | 技能描述笼统，与岗位匹配度低 |
| 0-39 | 不足 | 无技能描述或技能与岗位完全无关 |

### 资质证书（权重10%）

| 得分区间 | 评级 | 判断标准 |
|:---:|:---:|:---|
| 85-100 | 优秀 | 有行业权威证书，外语能力强 |
| 70-84 | 良好 | 有相关行业证书，外语能力良好 |
| 55-69 | 中等 | 有基础证书，外语能力一般 |
| 40-54 | 较弱 | 证书较少或与岗位无关 |
| 0-39 | 不足 | 无任何证书或资质证明 |

### 简历逻辑性（权重10%）

| 得分区间 | 评级 | 判断标准 |
|:---:|:---:|:---|
| 85-100 | 优秀 | 职业发展路径清晰，能力递进明显，各模块内容一致 |
| 70-84 | 良好 | 职业发展较清晰，能力有一定递进 |
| 55-69 | 中等 | 职业发展一般，能力递进不明显 |
| 40-54 | 较弱 | 职业发展混乱，能力无递进 |
| 0-39 | 不足 | 职业发展无逻辑，内容前后矛盾 |

### 简历合理性（权重10%）

| 得分区间 | 评级 | 判断标准 |
|:---:|:---:|:---|
| 85-100 | 优秀 | 经验年限与岗位匹配，职责合理，成果与角色匹配 |
| 70-84 | 良好 | 经验年限基本匹配，职责较合理 |
| 55-69 | 中等 | 经验年限一般，职责基本合理 |
| 40-54 | 较弱 | 经验年限偏短，职责与年限不匹配 |
| 0-39 | 不足 | 经验年限严重不足，职责与年限严重不匹配 |

### 简历衔接性（权重5%）

| 得分区间 | 评级 | 判断标准 |
|:---:|:---:|:---|
| 85-100 | 优秀 | 时间线完整无空缺，领域过渡自然 |
| 70-84 | 良好 | 时间线基本完整，领域过渡较自然 |
| 55-69 | 中等 | 有少量时间空缺（已解释） |
| 40-54 | 较弱 | 有明显时间空缺（未解释） |
| 0-39 | 不足 | 时间线混乱，领域频繁变动 |

## 评分计算方法

```
综合得分 = Σ(各维度得分 × 权重)
```

## 评分原则

1. **客观性**：基于简历中明确呈现的信息评分
2. **保守性**：信息不足时给予中等偏下评分
3. **建设性**：每个维度必须给出至少1条优点、1条不足、1条优化建议
4. **区分度**：评分需体现不同简历之间的质量差异
5. **独立性**：各维度评分相互独立

## 特殊情况处理

| 特殊情况 | 处理方式 |
|:---|:---|
| 应届生无工作经历 | 工作成果维度按50分基准评分 |
| 工作经历较少（<2年） | 项目业绩权重提升至30%，工作成果降至20% |
| 无项目经历 | 项目业绩维度按40分基准评分 |
| 无JD输入 | 评分不涉及岗位匹配度 |
| 有JD输入 | 增加岗位匹配度评估，但不纳入加权评分 |
